# Nile Visualizer Plan

Static planning microsite for the Cassini Nile simulator visualizer.

## Open locally

Open `index.html` directly in a browser, or serve the folder with a simple static server.

Example:

```powershell
cd C:\Users\danie\dev\cassini-nile-visualizer-plan\nile-visualizer-plan
python -m http.server 4173
```
